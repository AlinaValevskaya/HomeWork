Описание
Система тестирования реализована с помощью 3х классов:
- Test;
- Question;
- Option;

	Класс Option содержит описание объекта "Вариант ответа". 
	Заданы параметры:
	String answer;//текст ответа
    boolean correct;//переменная, определяющая правильный ответ
    Обращение к переменным реализовано через методы getAnswer() и getCorrect().

    Класс Question содержит описание объекта "Вопрос".
    Заданы параметры:
    String question;//текст вопроса
    Option[] option;//массив вариантов ответа
    Обращение к переменной question реализовано через метод getQuestion().

    Класс Test включает алгоритм реализации механизма тестирования.
    В качестве примера заданы 2 вопроса и 3 варианта ответа к каждому вопросу.
    Вывод вопроса на консоль происходит в случайном порядке через цикл.
    Метод calculate() выводит текст вопроса и варианты ответа, принимает ответ пользователя и сравнивает со значением переменной correct. 
    Количество правильных ответов пользователя фиксируется в переменной s.
    Так же реализована фиксация времени начала и завершения тестирования.

В качестве результата пользователь получает количество правильных ответов и время, потраченное на тестирование.
